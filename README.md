# Portfolio pessoal
Este projeto consiste em um portfólio pessoal desenvolvido para apresentar informações profissionais, habilidades técnicas, projetos e formas de contato de maneira organizada e responsiva.

A aplicação foi construída utilizando apenas tecnologias web nativas, com foco na prática de desenvolvimento front-end e na criação de interfaces modernas e intuitivas. O projeto também foi utilizado como atividade acadêmica da disciplina de **Aplicações Web e Móveis**.

## Tecnologias Utilizadas
- **HTML**
- **CSS**
- **JavaScript**

## Funcionalidades
- **Página inicial de apresentação**
- **Seção de formação**
- **Seção de projetos**
- **Seção de contato**
