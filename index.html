Meus projetos em destaque

Tela de Login

Aplicação desenvolvida para simular o processo de autenticação de usuários em um sistema. O projeto apresenta uma interface simples e intuitiva para entrada de credenciais, focando na experiência do usuário e na construção de telas utilizadas no dia a dia de aplicações web. Seu principal objetivo foi praticar conceitos de desenvolvimento de interfaces, organização visual e estruturação de páginas voltadas para autenticação de usuários.

Gymlog

GymLog é uma aplicação desenvolvida para auxiliar no acompanhamento e gerenciamento de treinos de academia. O sistema permite registrar treinos realizados, acompanhar exercícios, organizar fichas de treino e manter um histórico das atividades, facilitando o controle da evolução dos usuários. O projeto foi criado com foco em oferecer uma experiência prática para o gerenciamento de rotinas de treino, aplicando conceitos de desenvolvimento backend, organização de dados e construção de APIs.

💻

Library manager

LibraryManager é um sistema de gerenciamento de biblioteca desenvolvido para facilitar o controle e a organização de livros. A aplicação permite cadastrar, consultar, editar e gerenciar informações do acervo, simulando o funcionamento básico de uma biblioteca. O projeto foi criado com o objetivo de praticar o desenvolvimento de sistemas utilizando Java, aplicando conceitos de organização de dados e lógica de programação em um cenário real.
